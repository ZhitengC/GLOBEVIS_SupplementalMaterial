# Annotations Folder

## Organization
<br/>

- [Annotations_Full](Annotations_Full) - full annotation release.
  - 34 locale JSON files.
  - 5,180 annotated records in total.

<br/>

- [schema.md](schema.md) - schema reference for record fields, nested sections, and value conventions.

## Record Characteristics

- Each record includes `image`, `source`, and a structured `meta` object.
- Annotation dimensions include chart type, writing/reading direction, number formatting, datetime formatting, units, color semantics, axis/layout semantics, and extracted text.
- Filenames are stable and aligned with image payload naming in the `images` folder.
